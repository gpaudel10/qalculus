# Public images and screenshots

This directory is reserved for approved visuals made with synthetic or
otherwise public-safe material.

Before publishing an image, check that it:

- supports a specific public statement;
- is labelled honestly as a screenshot, mock-up, or illustration;
- contains no credentials, private URLs, addresses, hostnames, email addresses,
  customer information, or internal identifiers;
- reveals no private source code, infrastructure view, console, or unpublished
  method;
- has safe browser tabs, window titles, notifications, and file metadata; and
- includes enough context to explain what the viewer is seeing.

We will publish product screenshots only after a fresh, repeatable run using an
approved synthetic fixture. Until a future system has a working prototype,
visuals for it must be clearly labelled as conceptual illustrations.
